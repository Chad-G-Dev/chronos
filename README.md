# CHRONOS

`Chronos` is a cli application for tracking time spent doing things.

## Installation

1. Download the chronos repertory on your computer
2. Open the chronos directory.
3. run `cargo install --path .` (make sure cargo is installed).

## Usage

### List all available commands

In your terminal type `chronos help`

### Tracker creation

In your terminal type `chronos create [tracker name]`

### Start or stop the tracker

In your terminal type `chronos toggle [tracker name]`

### List all trackers

In your terminal type `chronos list`

### Get tracker time

In your terminal type `chronos log [tracker name]`

### Get chronos version

In your terminal type `chronos --version`
