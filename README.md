# CHRONOS

`Chronos` is a cli application for tracking time spent doing things.

## Installation

### Option A: Via the installer program.

1. Download the chronos repository on your computer.
2. Go in the installers directory and run the one for your operating system.

### Option B: Via cargo

1. Download the chronos repository on your computer.
2. Open the chronos directory.
3. run `cargo install --path .` (make sure cargo is installed).

## Usage

### Run the tui
In your terminal type `chronos`

### List all available commands

In your terminal type `chronos help`

### Tracker creation

In your terminal type `chronos create [tracker name]`

### Start or stop the tracker

In your terminal type `chronos toggle [tracker name]`

### List all trackers

In your terminal type `chronos list`

### Get tracker time

In your terminal type `chronos report [tracker name] [day amount in number or 'all']`

### Open Tui
In your terminal type `chronos`
